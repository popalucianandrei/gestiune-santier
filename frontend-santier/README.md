# Frontend Gestiune Șantier

Frontend React + Vite pentru aplicația ta de gestiune șantier.

## Cerințe

- Node.js instalat
- backend-ul pornit pe `http://localhost:3001`

## Pornire

```bash
npm install
npm run dev
```

Apoi deschizi în browser:

```text
http://localhost:5173
```

## Important

Backend-ul trebuie să rămână pornit în paralel:

```bash
node server.js
```

## Funcții incluse

- dashboard
- șantiere: citire + adăugare
- echipe: citire + adăugare
- furnizori: citire + adăugare
- materiale: citire + adăugare
- proiecte: citire + adăugare
- consum stoc: citire + adăugare
- utilaje: citire + adăugare
- alocări utilaje: citire + adăugare
- tab separat „Adăugare date” pentru introducere rapidă

## Notă

Am refăcut structura ca să nu mai piardă focusul la tastare.
